Satellite Image Classification using CNN, Grad-CAM, and BLIP

About the Project

This project applies convolutional neural networks (CNN) to classify satellite images from the EuroSAT dataset into 10 land use classes. Grad-CAM and BLIP are used to enhance interpretability.

Dataset Used

We used the EuroSAT RGB Dataset containing 27,000+ labeled satellite images from Sentinel-2. It can be accessed at:
https://github.com/phelber/eurosat
